# TODO Application - Spring Boot

## Features

- CRUD operations
- DTO pattern with validation
- Layered architecture
- Logging using SLF4J
- Notification service simulation
- Unit testing using JUnit & Mockito

## Tech Stack

- Java 17
- Spring Boot
- JPA (Hibernate)
- H2 Database
- Maven

## APIs

- POST /todos
- GET /todos
- GET /todos/{id}
- PUT /todos/{id}
- DELETE /todos/{id}

## Logging

Implemented using SLF4J in Controller and Service.

## Testing

Unit testing done using JUnit and Mockito.

## Notification

Simulated using NotificationServiceClient.
